## About Me
我是一名用户研究方向的从业者，专注于消费者洞察、品牌认知和行为分析。在过去的项目中，我主导了多项行业（母婴奶粉、饮品、快餐）品牌用户研究项目。
## Project 1: 宝妈群体在奶粉选择和购买中的用户行为分析（已脱敏）
### 研究背景
在母婴电商平台中，奶粉往往被宝妈视为高风险品类，一旦选错就很难回头。既涉及安全风险，又伴随强烈的情绪压力。因此用户在做出购买决策之前，通常会反复权衡和比较大量信息。所以广告内容在其中承担“降低不确定性”的角色。因此有必要对该场景下的用户体验进行拆解与理解。本分析基于真实项目访谈数据，聚焦于不同阶段宝妈，尝试理解他们如何感知奶粉品牌广告，并将其转化为信任或犹豫。
### 研究对象与方法
研究对象：基于真实项目访谈资料整理的宝妈群体样本（覆盖不同年龄与育儿阶段）本研究样分布广泛，包含low tier city和up tier city，即所呈现的决策心理机制具有跨区域可迁移性。
研究方法：半结构式深度访谈
围绕，
1.奶粉品牌的第一印象
2.对本品牌 vs 竞品广告的记忆与感受
3.本品广告卖点认知
4.广告内容在决策中的作用
本分析聚焦用户在决策过程中的心智与体验感知，不直接对销量结果作因果推断。
### 用户心智与信息推演路径
访谈显示，宝妈在接触奶粉广告信息后，并非直接形成判断，而是经历一个逐层推演的过程：首先，她们感知广告中的视觉与符号信息（如牧场、成分表和分子图等），随后基于自身关注点筛选信息，并结合过往经验对其赋予意义。在此基础上，宝妈进一步形成关于产品安全性与可靠性的推断。随着类似信息的反复出现，这种推断逐渐形成稳定的信念，并最终影响选择和购买行为。
### 核心用户差异
通过访谈内容对比可以观察到，宝妈并非一个同质人群，而是基于不同安全感来源形成对广告的差异反应，从而表现出差异化的行为决策模式。
表面上大部分宝妈关注点集中在奶源、安全与经验，但深层差异并不在于”关注什么“，而是她们如何获得安全感、以及决策依据为何。
基于本次访谈的归纳，宝妈在选择奶粉上，主要呈现三种安全感获取路径，从而形成不同的决策行为：
人群1：安全感导向型——通过外部信息获得安全感，责任外包
* 风险管理方式：通过权威、品牌规模和社会共识来降低不确定性
* 信息来源：专家、医生、国际标准
* 典型行为：换奶粉成本高，心理成本大于经济成本，对负面消息敏感度高
对人群1来说，广告更多的是在强调“这是一个被认可的正确选择”，从而降低在选择过程中的心理压力。

人群2：经验驱动型——不依赖广告判断，通过实际使用的结果反馈获得安全感，责任自持
* 风险管理方式：通过自身或者身边的人的实际喂养结果建立个人确定性。
* 信息来源：自身/身边朋友的经验，使用后的结果（消化、体重、上火等）
* 典型行为：对广告态度偏谨慎，对新概念反应弱，更关注真实使用反馈、同类宝妈经验
对人群2来说，广告不是她们决策的依据，其选择往往由实际喂养结果决定。即使出现与经验不一致的信息，也通常不会动摇其判断。

人群3：理性掌控型——通过理解和深入认知获得安全感，责任理性化
* 风险管理方式：通过理解成份、含量和作用机制，来获得掌控感
* 信息来源：配方信息，成分含量
* 典型行为：对情绪化表达敏感度低，更信任“信息密度高的内容”，更容易被逻辑感的差异说明信息说服，对模糊的表述警惕。
对人群3来说，广告是否有价值，取决于内容能否说明产品的独特之处，并给出可以理解的理由。

研究发现，在高风险消费决策中，用户分化的关键不在于人口属性的差异，而是风险认知方式和决策责任归因的不同。这一差异直接影响用户对广告信息、专业信息与成分机制说明的反应路径。
该人群划分方式可迁移至其他高风险决策场景，如母婴用品、医疗养生器械与教育服务等的选择（一次性，没办法反悔，做了就是做了且有实质性的影响）。
### 关键心理机制
在奶粉这一高风险品类中，广告更多承担的是解释和缓解焦虑的功能，帮助用户在复杂的信息中形成”我知道自己为什么这样选“的感觉，而非直接推动购买。
当广告内容与宝妈当前阶段的核心焦虑点匹配，或提供清晰、可验证的信息线索时，用户更容易产生一种“被支持做决定”的体验感受。
### 可验证假设
基于以上访谈洞察，本研究提出以下可进一步验证的假设：
* 不同育儿阶段的宝妈，对广告内容的偏好存在显著差异
* 广告中“专业背书”的作用更多体现在降低决策焦虑，而非直接提升购买意愿
### 研究价值
该洞察可用于改进：
* 电商详情页内容侧重点
* 内容平台种草信息的表达方式
* 不同类型用户的差异化沟通策略
